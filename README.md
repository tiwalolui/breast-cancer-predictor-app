#  Breast Cancer Prediction App

A machine learning web app that predicts whether a breast mass is *malignant* or *benign* based on clinical measurements.

## 🔗 Live Demo
[Click here to try the app](https://share.streamlit.io/user/tiwalolui)

##  About
This app uses a *Logistic Regression* model trained on the Breast Cancer Wisconsin dataset from Scikit-learn. It takes 5 key measurements as input and predicts the likelihood of malignancy.

##  Model Details
- *Algorithm:* Logistic Regression
- *Dataset:* Breast Cancer Wisconsin (Scikit-learn)
- *Accuracy:* 97.3%
- *Features used:* Mean Radius, Mean Texture, Mean Concavity, Worst Radius, Worst Concavity

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Streamlit
- NumPy
- Pickle
